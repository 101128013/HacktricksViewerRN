# Theme Preview Images

This directory contains visual preview images for each theme. These images are used in the theme selector to show users what each theme looks like.

## Themes Available

- **Dark**: Classic dark theme with blue accents
- **Ocean**: Deep blue theme inspired by ocean depths
- **Forest**: Natural green theme with earthy tones
- **Sunset**: Warm orange and purple theme
- **Purple**: Rich purple theme with vibrant accents
- **Matrix**: Retro green terminal-inspired theme

## Preview Format

Each theme preview shows:
- Background color (main app background)
- Surface color (cards, panels, dialogs)
- Primary color (buttons, links, active states)
- Accent color (highlights, secondary actions)
- Sample text and UI elements

## Accessibility

All theme previews support high contrast mode, which can be toggled in the theme selector.

## Usage

These images are displayed in the ThemeSelector component to help users choose their preferred theme visually.
